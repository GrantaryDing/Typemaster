# TpMaster AI+

A **typing test and typing practice application** built with **React + TypeScript + Vite**.

TpMaster AI+ is more than a basic typing-speed counter. It is designed as a richer typing trainer with multiple modes, detailed feedback, theme support, and challenge-oriented practice. From the code structure, it also includes specialized modes such as IELTS-oriented practice and challenge gameplay elements.

## Project highlights

- multiple typing modes
- detailed typing engine and scoring logic
- theme system with several visual styles
- challenge modes
- IELTS-oriented practice categories
- progress and performance tracking features
- modern frontend stack with React and TypeScript

## Notable features visible in the app structure

Based on the current codebase, the app includes or supports:

- time-based typing tests
- word-count-based typing tests
- IELTS writing-related categories
- challenge modes such as speed listening / speed typing
- theme presets like light, dark, matrix, sunset, ocean, cyberpunk, and coffee
- custom typing engine logic
- multiplayer-related code hooks
